Analysis Skills
● Built
ai-seo-aeo

Page-level audit

Audits a specific page against AI search best practices

  • Content structure and extractability
  • Schema markup
  • AI bot access (robots.txt)
  • External presence (G2, Wikipedia, roundups)
  • Competitive citation gaps
Produces → Page audit HTML report
● Built
llm-visibility-analyzer

Brand-level visibility analysis

Analyzes completed tracker data to produce a strategic visibility report

  • Citation rates by topic, platform, intent type
  • Blackout category diagnosis
  • Competitor frequency analysis
  • Ranked content gap recommendations
Produces → LLM visibility HTML report
Outputs
Page Audit Report

On-page + off-page action plan

HTML report. What to fix on the page, what to fix externally. P1/P2/P3 prioritized with owners and effort estimates

  • Content rewrite brief
  • Schema implementation checklist
  • Bot access verification
  • Third-party presence gaps
  • Competitive content gaps
LLM Visibility Report

Strategic content gap brief

HTML report. Where Applause is invisible, who owns those spaces and the specific content arguments that would close each gap

  • 17% baseline citation rate (June 2026)
  • 4 complete blackout categories
  • Testlio cross-category competitive threat
  • 0% evaluative/investigative intent rate
  • 6 ranked content recommendations

How the Skills Connect

Run the LLM visibility analyzer first to identify which topics and categories need attention. Then run the page-level audit on each priority page to get the implementation brief.

Strategy sets the direction. The audit builds the brief.
Page-level Audit Answers: how do we fix this specific page?
LLM Visibility Analyzer Answers: where are we invisible and why?
