The Five Building Blocks

How Claude is organized

Chat

Quick questions, research, one-off drafts, and brainstorming. Works like any other AI — but smarter when you've built out your custom instructions.

Start a new chat when the topic shifts. Long threads burn more tokens and drift off-track.

Projects

A dedicated space that builds memory over time. The more you use it, the better it knows your context, your voice, and your work history.

One person or topic per project. Sharing a project shares instructions and files — not your individual chat history.

Skills

Portable, reusable workflows. They run the same way every time and carry no memory between uses. Pull them into any chat or project with a /slash command.

Skills encode how to do something. Projects hold what to know. They work best together.

Artifacts

A gallery of everything you've made — HTML files, landing pages, documents, designs. No more digging through old chats to find something you built.

Click any artifact to jump straight back to the conversation where it was created.

Connectors

Links Claude to your other tools. You set what each connector can and can't do — for example, let it read email without allowing it to send.

Available now: Gmail, Google Calendar. Ask IT about GitHub — that one unlocks shared skill repositories.

Models & Settings

Pick the right model for the job

Default

Sonnet

Use this for most of what you do — writing, research, campaign work, prompting. Fast and capable. Keeps token usage lower.

Heavy Lifting

Opus 4.8

Complex analysis, long documents, nuanced strategic work. Move up to this when Sonnet isn't giving you enough depth.

Unavailable

Fable 5

On federal hold. For what we're building, you wouldn't need it — it's designed for complex cybersecurity work and costs significantly more.

Usage

Staying in bounds

$200
per person, per month
That's plenty. Dana is the team's most active user and hit $50 by mid-month — well within budget.

What doesn't show token usage

  • Chat has no live ticker. Learn by feel — model selection and prompt length are the main levers.
  • Cowork also has no ticker. Build the habit of running lean before you need it.

What does show token usage

  • Claude Code — shows a real-time message on token consumption. If you have Code access, that's your best feedback loop.
  • Settings → Usage — your total monthly spend, updated in near real time.

Quick tips

  • Check your usage once a day once you're active — Settings → Usage
  • Match model to task — Sonnet for most things, Opus only when you need it
  • Start a new chat when the topic shifts — long threads compound the cost
  • Be concise — more prompt isn't always better
  • Heavy PDF conversions (like large decks to Markdown) are expensive — use Gemini for those
